Schedule Scraper

A small python program that pulls the HTML code from the online class catalog for Fullerton College. Using Beautiful Soup 4 and Gspread I made an algorithm to pull data from each table element and organize it based on column number and data types. Once the Data is pulled it then sends all the data to a Google Spreadsheet for cloud access at any time.

The purpose behind this program was to make choosing classes easier for me, it filters through classes that are Restricted or waitlisted and only showed those that were either open or closed (so I could see which classes recently closed)


#--Algorithm snippet for sifting through values--#
for tr in soup.find_all('tr'):
    if stop == True:
        break
    if len(tr) < 15:
        continue
        
    #--Look for all data entries and skip any that contain...----#
    #--RESTRICTED or WAITLISTED-(Keeping Closed to stay updated)-#
    #----OPEN:--Open for Enrollment------# 
    #----CLOSED:--Closed for Enrollment--#
    #----RESTRICTED:--Special Classes----#
    #----Waitlisted:--Full/Open Waitlist-#
    
    for td in tr.find_all('td'):
        if td.text == "OPEN" or td.text == "CLOSED":
            DataCheckPoint = True

        if DataCheckPoint == True:
            #--Skipping initial intro rows--#
            if td.text == "End of report":
                stop = True
                break
            if td.text == "Status":
